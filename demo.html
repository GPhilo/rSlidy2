Basics

  • Each presentation in a single XHTML file
  • Each slide is enclosed in <div class="slide"> ... </div>
  • Use regular markup within each slide
  • Include style sheet (rslidy2.css) and script (rslidy2.js) in the header
  • You may include a seperate stylesheet for you presentation if you want to change the default style
    • See demo.css as an example

Title Page

  • To create a title page use <div class="slide titleslide"> ... </div>

Incremental Display of Slide Contents

  • For incremental display, use class="incremental" when creating an HTML-list
  • Like: <ul class="incremental"> ... </ul>

Add Images to your Slide-Deck

  • Use the HTML <img>-Tag
  • Include the image in your images/ folder
  • Here's the code for adding image: example.png: <img src="images/example.png">

Anything else is just basic HTML and CSS

  • Right now it is strongly recommended to use a recent version of Chrome to display the slides
