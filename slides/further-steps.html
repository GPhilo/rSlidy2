Incremental Display of Slide Contents

  • For incremental display, use class="incremental" when creating an HTML-list
  • Like: <ul class="incremental"> ... </ul>

Anything else is just basic HTML and CSS

  • Right now it is strongly recommended to use a recent version of Chrome to display the slides
  • Report issues on our github page: https://github.com/GPhilo/rSlidy2
  • But please note that this is a work in progress
  • Have fun!
