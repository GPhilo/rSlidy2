Basics

  • Each presentation in a single XHTML file
  • Each slide is enclosed in <div class="slide"> ... </div>
  • Use regular markup within each slide
  • Include style sheet (rslidy2.css) and script (rslidy2.js) in the header
  • You may include a seperate stylesheet for you presentation if you want to change the default style
    • See demo.css as an example
